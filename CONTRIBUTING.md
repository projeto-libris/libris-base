# Contribuindo com o Libris Base

Obrigado por considerar contribuir com o **Libris Base**!

O objetivo deste projeto é fornecer uma base simples e acessível para a criação de projetos da **família de enciclopédias gerais Libris**.

Toda contribuição é bem-vinda, seja ela uma correção de erros, melhoria de código, documentação ou novas funcionalidades.

## Como contribuir

### Relatando problemas

Se você encontrou um erro, abra uma *issue* e informe:

* O que aconteceu;
* Como reproduzir o problema;
* O comportamento esperado;
* Capturas de tela, se aplicável;
* Informações do navegador ou ambiente utilizado.

## Sugerindo melhorias

Você também pode abrir uma *issue* para:

* sugerir novos recursos;
* propor mudanças na interface;
* melhorar a documentação;
* discutir ideias para o projeto.

Isto não poderá incluir **alteração do design do site**.

## Enviando alterações

1. Faça um *fork* deste repositório;
2. Crie uma nova branch:

```bash id="1d4y2u"
git checkout -b minha-alteracao
```

3. Faça suas alterações;
4. Salve e teste as mudanças;
5. Envie os commits:

```bash id="jv6r9w"
git commit -m "Descrição da alteração"
```

6. Envie a branch:

```bash id="6l1pbi"
git push origin minha-alteracao
```

7. Abra um *Pull Request*.

## Diretrizes

Ao contribuir, procure:

* manter o código simples e legível;
* seguir o estilo já utilizado no projeto;
* documentar novas funcionalidades;
* evitar alterações que quebrem a compatibilidade sem necessidade.

## Documentação

Melhorias na documentação são tão importantes quanto melhorias no código.

Você pode contribuir:

* corrigindo erros;
* atualizando tutoriais;
* adicionando exemplos;
* traduzindo conteúdos.


## Obrigado!

Cada correção, sugestão ou melhoria ajuda a tornar o Libris Base uma enciclopédia geral melhor para todos.

📚 Construa conhecimento. Compartilhe conhecimento. Preserve conhecimento.
