# Libris Base

**Libris Base** é o código-fonte oficial utilizado para a criação de projetos da **família de enciclopédias gerais Libris**.

O projeto fornece uma base simples para a construção de enciclopédias digitais independentes, voltadas à documentação, preservação e divulgação do conhecimento.

**AVISO: Projetos Libris que incluirem apenas `© 2026 Projeto Libris. Todos os direitos reservados.` são oficiais.**

## O que é um projeto Libris?

Um projeto Libris é uma enciclopédia digital criada a partir do Libris Base e dedicada a um tema específico, como:

* uma cidade;
* uma região;
* uma instituição;
* um projeto;
* uma comunidade;
* qualquer outro assunto de interesse público.

Exemplos:

* Libris para Alunos
* Libris para Programadores
* Libris Educação
* Libris Municipal

## Recursos

* Interface simples e leve;
* Fácil personalização;
* Suporte a categorias e artigos;
* Estrutura estática;
* Código-fonte aberto para adaptação.

## Como começar

1. Faça um fork deste repositório;
2. Clone o projeto:

```bash
git clone https://github.com/projeto-libris/libris-base.git
```

3. Personalize:

* nome;
* descrição;
* logotipo;
* cores;
* categorias.

4. Publique o seu projeto no serviço de hospedagem de sua preferência **(recomendado: Cloudflare Pages)**.

Consulte o tutorial completo em **[TUTORIAL.md](./TUTORIAL.md)**.
